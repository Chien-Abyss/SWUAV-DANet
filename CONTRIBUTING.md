# 贡献指南

欢迎提交 Issue 或 PR 改进 SWUAV-DANet。

## 开发环境

1. 建议使用虚拟环境 `python -m venv .venv && .\.venv\Scripts\activate`
2. 安装依赖：`pip install -r requirements.txt && pip install -e .`
3. 运行示例训练/推理脚本，确认本地环境可复现。

## 提交规范

- 代码保持 PEP8 / ruff 友好，必要时补充注释。
- 新功能请带上最小的使用示例或测试（如新增脚本的 `--help` 能正常运行）。
- 请在 PR 描述中说明改动目的、影响范围以及验证方式。

## 行为准则

保持友好、专业的交流，尊重社区成员。
